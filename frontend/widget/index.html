AI Support Agent Widget

This page demonstrates the embeddable chat widget. Click the chat bubble in the bottom-right corner to open it.

How to embed

Add this script tag to any HTML page:

<script
  src="http://localhost:8000/widget/widget.js"
  data-api-url="http://localhost:8000"
  data-tenant-id="default"
  data-api-key="demo-key"
  data-title="Support"
  data-color="#6B3AC6"
></script>

Configuration

Attribute Default Description
data-api-url http://localhost:8000 Backend API base URL
data-tenant-id default Tenant identifier
data-api-key demo-key Tenant API key for auth token exchange
data-title Support Chat header title
data-color #6B3AC6 Accent color (hex)
data-greeting Hi! How can I help you today? Initial bot message
